## Требования к системе

Перед сборкой убедитесь, что у вас установлены:
1. **CMake** (версии 3.10 или выше)
2. **Компилятор C++** с поддержкой стандарта C++17 (GCC, Clang или MSVC)

---

## Инструкция по сборке и запуску

### Сборка на Linux

```bash
# 1. Создание и переход в папку сборки
mkdir build && cd build

# 2. Генерация файлов сборки
cmake ..

# 3. Компиляция проекта
cmake --build .

# 4. Запуск игры
./rpg_game
```

### Сборка на Windows

#### Вариант 1: Через командную строку (cmd / PowerShell)
Убедитесь, что CMake и ваш компилятор (например, MinGW) добавлены в переменные среды (PATH).

```powershell
# 1. Создание и переход в папку сборки
mkdir build
cd build

# 2. Генерация файлов сборки
cmake ..

# 3. Компиляция проекта
cmake --build . --config Release

# 4. Запуск игры
.\Release\rpg_game.exe
```
*(Примечание: Если вы используете MinGW, исполняемый файл может появиться прямо в папке `build/rpg_game.exe`)*

#### Вариант 2: Через Visual Studio
1. Откройте **Visual Studio**.
2. Выберите **«Открыть локальную папку»** (Open a local folder) и укажите корень репозитория.
3. Visual Studio автоматически обнаружит `CMakeLists.txt` и настроит проект.
4. Нажмите **F5** для сборки и запуска.
