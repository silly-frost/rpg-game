## Инструкция по сборке и запуску

Для компиляции проекта вам понадобятся компилятор с поддержкой **C++17** (или выше) и утилита **CMake**.

### Запуск на Linux (Ubuntu/Debian/Arch)

1.  Откройте терминал и перейдите в корень проекта:
    ```bash
    cd /home/frost/projects/rpg-game
    ```
2.  Создайте папку для сборки и перейдите в неё:
    ```bash
    mkdir build && cd build
    ```
3.  Сконфигурируйте проект с помощью CMake (он автоматически подготовит `map.json` в папке сборки):
    ```bash
    cmake ..
    ```
4.  Скомпилируйте проект:
    ```bash
    make
    ```
5.  Запустите игру:
    ```bash
    ./rpg_game
    ```

### Запуск на Windows (Command Prompt / PowerShell)

#### Способ 1: Через MinGW (GCC) в консоли
1.  Откройте командную строку (CMD) и перейдите в папку проекта:
    ```cmd
    cd C:\projects\rpg-game
    ```
2.  Создайте папку сборки:
    ```cmd
    mkdir build
    cd build
    ```
3.  Сгенерируйте файлы сборки для MinGW:
    ```cmd
    cmake -G "MinGW Makefiles" ..
    ```
4.  Скомпилируйте проект:
    ```cmd
    cmake --build .
    ```
5.  Запустите игру:
    ```cmd
    rpg_game.exe
    ```

#### Способ 2: Через Visual Studio
1.  Откройте **Visual Studio**.
2.  Выберите пункт **"Открыть локальную папку"** (Open a local folder) и укажите корень вашего проекта `rpg-game`.
3.  Visual Studio автоматически распознает файл `CMakeLists.txt` и настроит проект.
4.  Нажмите кнопку **"Уведомление о запуске / Выбрать целевой объект"** (Зелёная стрелочка сверху) и выберите `rpg_game.exe`. Проект скомпилируется и запустится сам.

## Структура Репозитория

```text
├── CMakeLists.txt        # Сценарий сборки CMake
├── map.json              # База данных локаций игрового мира
├── README.md             # Документация проекта
├── headers/              # Заголовочные файлы (.hpp)
│   ├── json.hpp          # Библиотека nlohmann/json для парсинга файлов
│   ├── map.hpp           # Объявления карты и переходов
│   ├── npc_data.hpp      # Сценарии диалогов, торговцев и квестов
│   ├── player.hpp        # Структура персонажа и функции сохранения
│   └── utils.hpp         # Общие утилиты (очистка экрана, пауза, структура Monster)
└── src/                  # Исполняемый код (.cpp)
    ├── main.cpp          # Главное меню игры и стартовая точка
    ├── map.cpp           # Логика загрузки JSON и перемещений с задержкой времени
    └── player.cpp        # Реализация статов, инвентаря и Save/Load системы
```
